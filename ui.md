# Fielding Set – UI Blueprint

## Design Style

Theme: Dark mode

Primary colors:
- background: #0f0f0f
- text: white
- accent: yellow or neon green

Typography:
- bold headings
- simple sans-serif font

The UI should be image-focused and meme friendly.

---

# Pages

Title:

FIELDING SET

Upload someone.

## Feed Page

Display posts in vertical card layout.

Each card includes:

- Image
- Name
- FIELDING SET label
- Reaction buttons
- Comment count

Cards should be large and easy to scroll.
